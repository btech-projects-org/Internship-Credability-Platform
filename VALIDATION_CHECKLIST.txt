┌──────────────────────────────────────────────────────────────┐
│ PHASE 8: USER SUPPORT                                        │
└──────────────────────────────────────────────────────────────┘

If user reports "app won't start":
☐ Check Windows 10/11 (64-bit requirement)
☐ Ensure 500 MB free space
☐ Try: Run as Administrator
☐ Check: Port 5000 not in use (netstat -an | find ":5000")

If user reports "browser doesn't open":
☐ Manual: Open http://localhost:5000 in browser
☐ Or: Manually navigate to frontend/pages/index.html

If user reports "predictions always same":
☐ Stub models are in use (expected for demo)
☐ Replace with real models for production

If user reports "antivirus warning":
☐ Safe to proceed (bundled exe triggers false positives)
☐ Whitelist in antivirus settings
☐ (Optional) Get code signing certificate for permanent fix
