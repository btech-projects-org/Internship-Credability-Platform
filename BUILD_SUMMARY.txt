└─────────────────────────────────────────────────────────────────┘

⚠ Windows-only (currently)
  ├─ Reason: PyInstaller bundles Windows-specific dependencies
  ├─ Impact: Cannot run on macOS/Linux
  └─ Solution: Create separate build.sh for macOS/Linux (future)

⚠ Large file size (~300 MB)
  ├─ Reason: Includes Python runtime + ML libraries
  ├─ Impact: Longer download time
  └─ Benefit: Eliminates dependency installation (saves time overall)

⚠ Antivirus may flag it
  ├─ Reason: Bundled executables sometimes trigger alerts
  ├─ Impact: User may see "suspicious file" warning
  └─ Solution: Code signing certificate (optional, $300/year)

⚠ Updates require full rebuild
  ├─ Reason: Each version is a new .exe
  ├─ Impact: User downloads new .exe each time
  └─ Benefit: No dependency version conflicts
